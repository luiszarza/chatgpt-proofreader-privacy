ChatGPT Proofreader Privacy Policy

Effective date: June 5, 2026

ChatGPT Proofreader is a browser extension that helps users proofread, rewrite, transform, and improve text that they intentionally select on webpages.

Summary

ChatGPT Proofreader does not collect, sell, rent, or store user data on the developer's servers. The extension only handles data needed to perform its single purpose: transforming user-selected text when the user requests it.

User Data Handled by the Extension

The extension may handle the following user data:

  • Selected webpage text: text that the user intentionally highlights and submits by clicking the Proofread button or the "Proofread with ChatGPT" context menu option.
  • OpenAI API key: the API key entered by the user in the extension options page so the extension can call the OpenAI API.
  • Extension settings: the selected model, action prompt, and display preferences configured by the user.

How User Data Is Used

User-selected text is used only to generate the proofreading, rewriting, or text transformation result requested by the user. The OpenAI API key is used only to authenticate requests to the OpenAI API. Extension settings are used only to control how the extension behaves for the user.

The extension does not use user data for advertising, analytics, profiling, creditworthiness, lending, or any unrelated purpose.

How User Data Is Handled

The extension runs in the user's browser. When the user submits selected text, the extension sends that selected text directly from the user's browser to the OpenAI API over HTTPS. OpenAI returns a transformed text result, which the extension displays in an on-page panel.

The developer does not receive, view, collect, or store the selected text or API key on developer-controlled servers.

Storage and Retention

The user's OpenAI API key, selected model, action prompt, and display preferences are stored in Chrome or Edge extension storage on the user's browser profile. These settings remain stored until the user changes them, clears browser extension storage, or uninstalls the extension.

The extension does not store selected webpage text after the requested proofreading or transformation result is displayed.
