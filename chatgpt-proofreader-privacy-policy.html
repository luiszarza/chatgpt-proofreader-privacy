ChatGPT Proofreader Privacy Policy

Effective date: May 29, 2026

ChatGPT Proofreader is a browser extension that helps users transform, proofread, and improve selected text on webpages.

Data We Do Not Collect

We do not collect, sell, rent, or store user data on our own servers. We do not track browsing history, page visits, clicks, location, or user activity for analytics or advertising.

Data Processed by the Extension

The extension only processes text that the user intentionally selects and submits by clicking the Proofread button or the "Proofread with ChatGPT" context menu option.

The selected text is sent directly from the user's browser to the OpenAI API so that OpenAI can return a transformed or corrected version of the text. The developer does not receive or store the selected text.

OpenAI API Key

The extension requires the user to enter a valid OpenAI API key. The API key is stored in the user's browser storage using Chrome or Edge extension storage. The developer does not receive or store the API key on external servers.

Third-Party Services

The extension sends user-selected text to the OpenAI API only when the user requests proofreading or text transformation. Use of the OpenAI API is subject to OpenAI's own terms and privacy policies.

Data Sharing

We do not sell user data. We do not share user data with third parties for advertising, profiling, creditworthiness, or unrelated purposes.

User Control

Users control what text is submitted. The extension does not automatically submit page content. Users may remove their API key or change extension settings at any time from the extension options page.

Contact

For questions about this privacy policy, contact the extension publisher.
