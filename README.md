# ChatGPT Proofreader Privacy Policy

This repository hosts the privacy policy for **ChatGPT Proofreader**, a browser extension that helps users proofread and improve selected text on webpages.

The published privacy policy is available through GitHub Pages once enabled for this repository.

## Extension Data Use

ChatGPT Proofreader does not collect, sell, rent, or store user data on the developer's servers.

The extension only processes text that the user intentionally selects and submits by clicking the Proofread button or the "Proofread with ChatGPT" context menu option.

The selected text is sent directly from the user's browser to the OpenAI API so that OpenAI can return a corrected or transformed version of the text.

The user's OpenAI API key is stored only in the user's browser extension storage and is not stored by the developer on external servers.
